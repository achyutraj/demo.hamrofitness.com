## About HamroFitness 

HamroFitness is a web-based application written in [Laravel-8](https://laravel.com/docs/8.x) (PHP framework) which is designed for gym owners to assist them to manage their business. 

## Installation Requirement
- [PHP >= 7.4](https://www.apachefriends.org/download.html).
- [Composer](https://getcomposer.org/) for dependencies.

## Major Features
- User,Role and Permission
- Branch Management
- GymClient and its subscription Management
- Report Generators
- SMS and Email Alert

### Dependencies Used 
- [barryvdh/laravel-dompdf](https://github.com/barryvdh/laravel-dompdf) for PDF generator
- [intervention/image](https://github.com/Intervention/image) for Image
- [guzzlehttp/guzzle ](https://laravel.com/docs/8.x/http-client) for HTTP Client Requests
- [maatwebsite/excel](https://github.com/SpartnerNL/Laravel-Excel) for Excel Exports
- [spatie/laravel-permission](https://spatie.be/docs/laravel-permission/v4/introduction) for Role and Permission
- [yajra/laravel-datatables-oracle](https://yajrabox.com/docs/laravel-datatables/master/installation) for DataTables
- [simplesoftwareio/simple-qrcode](https://www.simplesoftware.io/#/docs/simple-qrcode) for QR Code Generator
- [milon/barcode](https://github.com/milon/barcode) for BarCode Generator
- [laravel/sanctum](https://laravel.com/docs/8.x/sanctum) for API Token used in GymClient
